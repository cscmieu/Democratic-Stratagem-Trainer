\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGameManager}{Game\+Manager}} class represents the game manager that controls the game flow. 

It manages the current playlist, stratagem, sequence progression, and provides methods to update the playlist, check user input, and switch to the next stratagem.
